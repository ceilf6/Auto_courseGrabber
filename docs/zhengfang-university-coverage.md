# 正方高校覆盖与兼容性记录

最后核验：2026-06-21。

本文件用于回答“哪些高校使用正方教务系统”以及“本脚本是否适用于我的学校”。这两个问题不能混为一谈：正方软件客户关系、学校当前教学管理平台与本脚本在具体选课页面上的可用性，是三个不同层次的事实。

## 证据等级

| 等级 | 含义 | 可否称为“本项目已实测” |
| --- | --- | --- |
| A：正方教务平台已确认 | 2025–2026 年高校官方资料明确提到正方教学或选课平台。 | 不可以；仍需本项目的页面实测。 |
| B：正方官网公开高校客户 | 正方官网当前展示该校为高校客户。产品范围可能是教务、门户、学工、数据等任一模块。 | 不可以。 |
| C：本项目社区实测 | 有学校、系统版本或非敏感页面特征、脚本版本、测试日期和结果的报告。 | 可以，且必须附带报告链接。 |

不得把 A 或 B 自动升级为 C。不同学校可使用不同的正方产品、系统版本、页面定制和选课规则。

### 提交要求

- 可以提供：学校名称、省份、正方系统版本（若可见）、非敏感页面特征、脚本版本、测试日期、成功或失败结果、脱敏截图。
- 不要提供：账号、密码、验证码、Cookie、Token、学号、姓名、课程名单、请求包、内网地址或任何其他个人/学校敏感数据。
- 仅在学校规则允许的范围内测试；本项目不保证任何学校、任何学期或任何选课模块的兼容性。
